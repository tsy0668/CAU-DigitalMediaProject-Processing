Photoshop Program


- 상단의 메뉴들은 포토샵을 진행할 사진들입니다. 사진들 중의 하나를 클릭하면 포토샵을 진행합니다.
- 우측의 메뉴들은 구현한 기능들입니다.
	1. 흑백전환 - 사진을 흑백으로 변경합니다.
	2. hue 180degree - HSB모드에서 색조(H)만 변경합니다.
	3. inverse - 현재 사진의 RGB값을 정반대로 변경합니다.
	4. desaturation - 채도를 0으로 떨어뜨립니다.
	5. split - 선택한 사진을 9개로 출력합니다. 이 때 HSB의 색조(H)를 랜덤으로 출력합니다.
	6. shivering - 클릭한 상태를 유지하면 사진이 떨립니다.
	
	+ 카메라버튼을 누르면 현재 사진을 캡쳐합니다.
