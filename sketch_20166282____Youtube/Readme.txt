Makin YOUTUBE

- SpaceBar 입력 시 일시정지,진행을 반복할 수 있습니다.
- 방향키 좌,우 입력시 동영상 앞으로 10초 이동, 뒤로 10초 이동 가능합니다.
- 아래에 함수 부분을 설명하겠습니다.

bar() 
: 일정 위치에 마우스가 이동하면 회색 바를 생성합니다.

clock()
: 영상의 총 시간을 화면가로넓이에 맵핑합니다. 회색 바를 따라서 동영상의 
진행상황에 따라 빨간색 선이 이어져나가고, 마우스를 선 위에 올렸을 시에
그 시간의 영상화면과 시간을 보여주며 마우스 클릭 이벤트 발동 시 
해당 영상의 시간으로 이동합니다. 

playButton()
: 현재 영상의 진행시간을 text로 출력합니다. 버튼 클릭이나 스페이스바를 누르면
영상이 일시정지,진행을 반복합니다.(buttonFlag 전역변수 이용)

ward()
: 앞으로 10초 이동, 뒤로 10초 이동, 처음으로 되돌리기 버튼의 icon입니다.
마우스 클릭 이벤트 발동 시 해당 이벤트가 발생하며, 키보드의 좌,우 방향키 버튼으로도
동작합니다.

speedButton()
: speed를 조절하는 버튼을 만든 함수입니다. 숫자를 클릭하면 해당 배속의 속도로
동영상이 재생됩니다. 현재 배속을 나타내주기 위해 음영을 주었습니다.
