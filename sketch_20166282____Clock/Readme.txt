[20166282 컴퓨터공학과 4학년 양태성]

이 프로그램은 Reculsion과 Clock을 동시에 구현한 프로그램입니다.
draw() 함수가 실행됨과 동시에 reculsion이 시작되어 지속적으로
화면이 바뀝니다. (reculsion의 stack overflow는 해결했습니다.)
그래서 record기능을 하도록 구현하여 PDF를 제출하게 되었습니다.
허나 PDF파일 상에서 record 된 영상은 움직이지 않고 record를 마칠 당시의 화면만 출력됩니다.
제 노트북이 문제인지 모르겠으나 감안해주시면 감사하겠습니다.


-------함수설명---------

reculsion(15) 
 : 재귀함수 구현부분입니다.

clock()
 : 시간을 받아오고 시계의 모양을 구현하는 함수 입니다.

tex()
 : 화면 중앙에서 시간에 대한 text를 표시해주는 함수입니다.

lib()
 : 화면 중앙의 text 밑에 입술 모양이 있습니다. 디자인이 칙칙한 것 같아 넣어봤습니다.

KeyPressed()
 : record 기능 구현 함수입니다. 키보드 r키를 누르면 녹음이 시작되고 
   시작 된 상태에서 또 r을 누르면 녹음이 종료됩니다.
   q키를 누르면 레코드 기능을 중지합니다.
